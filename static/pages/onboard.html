Onboard a data source

Bring data in from a public API, an uploaded file, or a database connection. Every source runs through the same local pipeline — profile → transform → quality-check → document → load — landing as a queryable DuckDB table. Everything runs on this device. Once onboarded, a dataset can be evaluated for use in the US States Health Mart.
